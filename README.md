# AlipayHB

#### 项目介绍
支付宝红包收款码制作工具，网页版
Alipay red packets pay code

红包收款码：将红包码合并至收款码上。

顾客：既能领红包，又能付款；

商家：既能收款，又能收到红包返利。

单个网页工具：[https://scimence.gitee.io/alipayhb](https://scimence.gitee.io/alipayhb)
